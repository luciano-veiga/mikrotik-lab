# 🔴 MikroTik Lab PRO — Simulador Educacional

> Ambiente interativo estilo **Winbox** para aprender configurações reais de roteadores MikroTik de forma gamificada.

🌐 **Demo ao vivo:** [https://seuusuario.github.io/mikrotik-lab](https://luciano-veiga.github.io/mikrotik-lab/)

---

## 📋 Módulos disponíveis

| Módulo | Conteúdo |
|--------|----------|
| 🔌 Interface List | Tipos de interfaces, WAN, LAN, Virtual |
| 🌉 Bridge | Switch virtual, portas, PVID |
| 🏷️ VLAN | Segmentação, trunk, access, VLAN Filter |
| 📡 Wireless | AP Bridge, Virtual AP, WPA2, clientes |
| 🔗 PPPoE | Cliente e Servidor, Profiles, rate-limit |
| 📋 DHCP Server | Pools, leases, Make Static |
| 🛡️ Firewall / NAT | Filter rules, masquerade, dst-nat |
| 🔐 VPN L2TP/IPSec | Servidor, usuários, criptografia AES-256 |
| ⚡ QoS / Filas | Simple Queue, Queue Tree, PCQ, Burst |
| 🔒 Segurança | 9 boas práticas de hardening |
| 💻 Terminal CLI | +15 comandos RouterOS reais interativos |
| 🎯 Quiz PRO | 12 perguntas com feedback e pontuação |

---

## 💻 Terminal Interativo

Pratique comandos reais do RouterOS:

```
interface print
ip address print
ip route print
ip firewall filter print
queue simple print
ping 8.8.8.8
help
```

## 🚀 Como usar localmente

Basta abrir o arquivo `index.html` diretamente no navegador — sem dependências, sem servidor, sem instalação.

```bash
git clone https://github.com/seuusuario/mikrotik-lab.git
cd mikrotik-lab
# Abra index.html no navegador
```
